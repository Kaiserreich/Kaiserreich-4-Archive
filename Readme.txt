Thank you for downloading Kaiserreich for Hearts of Iron 4!
This version is designed for Hearts of Iron IV version 1.13.3.
It should be compatible with all hotfixes, but it isn't guaranteed.

Manual Installation Instructions
1) Download the mod
2) Delete any former versions
3) Extract the .zip to 'Documents\Paradox Interactive\Hearts of Iron IV\mod'
4) Make a copy of the `descriptor.mod` file in the folder, rename it to `Kaiserreich.mod` and put it in 'Hearts of Iron IV\mod'
  4.1) For versions prior to `0.16` "Hindsight is 2020": Move `Kaiserreich.mod` out of 'Hearts of Iron IV\mod\Kaiserreich' and into 'Hearts of Iron IV\mod'
5) Enable the mod in the Launcher mod menu and enjoy!
